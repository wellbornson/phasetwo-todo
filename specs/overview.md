# Project Overview

## Purpose
The goal of this project is to evolve a basic Todo application into a sophisticated AI-powered Personal Assistant Chatbot ("Rashid").
This documentation covers **Phase II: Todo Full-Stack Web Application**, which transforms the initial console application into a modern, multi-user web application with persistent storage and authentication.

## Current Phase: Phase II (Web App)
- **Objective**: Implement 5 Basic Level features (Task CRUD) as a responsive web application.
- **Focus**: Establishing the full-stack architecture, database integration, and security infrastructure (Authentication).

## Technology Stack
- **Frontend**: Next.js 16+ (App Router), TypeScript, Tailwind CSS.
- **Backend**: Python FastAPI.
- **ORM**: SQLModel.
- **Database**: Neon Serverless PostgreSQL.
- **Authentication**: Better Auth (with JWT plugins).
